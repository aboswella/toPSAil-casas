# Codex Task Protocol

Every task must use this structure.

## Task ID

Unique name.

## Goal

One sentence.

## Allowed files

Explicit list.

## Forbidden files

Explicit list.

## Source basis

Which source or doc justifies the task.

## Preconditions

What must already pass.

## Required tests

Exact commands.

## Stop conditions

When to stop instead of editing.

## Report format

Must include:

- files inspected;
- files changed;
- commands run;
- test results;
- unresolved uncertainties;
- validation impact;
- next smallest task.
