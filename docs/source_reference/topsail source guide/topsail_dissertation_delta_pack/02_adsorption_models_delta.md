## Implementation rules for future model work

1. **Capability manifest before model use.** Source: Diss p. 240-241 (PDF p. 275-276), §6.2.2. If a source paper names gL, MS-LDF, S-shaped isotherms, or hysteresis, emit `unsupported_model` until the actual model is implemented and tested.
2. **Do not conflate local nonlinear isotherm solves with a general implicit DAE framework.** Source: Diss p. 266-267 (PDF p. 301-302), §D.1.5; Diss p. 244 (PDF p. 279), §6.2.4. [INFERENCE] Multi-site Langmuir can be solved as a local nonlinear equilibrium subproblem, while MS-LDF may require dynamic implicit residuals.
3. **Dimensionless scaling is part of the model contract.** Source: Diss p. 165-166 (PDF p. 200-201), §3.7.1; Diss p. 259-268 (PDF p. 294-303), Appendix D. New submodels must expose dimensional and dimensionless code paths or a tested conversion wrapper.
4. **Schell integration should not be delayed by speculative adsorption-model work unless the current source model is unsupported.** Source: Diss p. 240 (PDF p. 275), §6.2.1 and §6.2.2. [INFERENCE] Current Schell failures should be diagnosed first through connectivity/accounting ledgers, as established in the earlier pack.
