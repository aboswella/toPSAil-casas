# Validation Reports

Generated validation reports belong here when a task explicitly asks for them.

Every report must include:

- model mode;
- parameter pack;
- source basis;
- grid/cell count;
- cycle count;
- CSS residual;
- runtime;
- hard failures;
- soft discrepancies;
- interpretation;
- unresolved uncertainties.

Large generated artifacts should remain untracked unless the task explicitly asks to commit them.
